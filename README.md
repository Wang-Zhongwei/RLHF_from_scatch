Build the entire Reinforcement Learning from Human Feedback pipeline on top of distilgpt2 from scratch, covering decoding, supervised fine-tuning, LoRA adapters, reward modeling, PPO, and modern preference-optimization alternatives like DPO, IPO, KTO, ORPO, and SimPO. The project culminates in evaluation tooling and a minimal chat interface so you can compare aligned and unaligned models end to end.
